1. What is Cloudflare?
-A Global Cloud Network: Cloudflare acts as a massive proxy shield between a website’s server and its visitors.

-Security & Performance: It protects websites from cyberattacks (like DDoS attacks) and speeds them up using a global Content Delivery Network (CDN).

-An All-in-One Toolkit: It provides DNS management, SSL certificates, web application firewalls (WAF), and serverless computing.

2.Domain, DNS, and How It Works
-Domain: The human-readable address of a website (e.g., google.com).

-Domain Name System (DNS): The "phonebook of the internet." Computers don't understand names; they understand numbers (IP addresses). DNS translates google.com into an IP address like 142.250.190.46.

How it works:

-You type a URL in your browser.

-Your computer asks a DNS Resolver for the IP.

-The resolver finds the Authoritative Nameserver for that domain.

-The IP is returned to your browser, and your browser loads the page.

3. Role of Cloudflare in the Internet Layer
-Reverse Proxy: Cloudflare sits right in front of your origin server at the application and network layers.

-Traffic Filter: All incoming traffic hits Cloudflare’s nearest data center first. Bad traffic (bots, hackers) is blocked, while legitimate traffic is passed through.

-Caching: Static content (images, HTML) is saved on Cloudflare's edge servers, reducing the load on your actual server.

4. How to Connect a Domain to Cloudflare & Manage DNS
-Add Site: Log into Cloudflare, click Add a Site, and type in your domain name.

-Scan Records: Cloudflare automatically scans and imports your existing DNS records.

-Change Nameservers: Cloudflare will provide you with two new nameservers (e.g., greg.ns.cloudflare.com).

-Update Registrar: Go to where you bought your domain (GoDaddy, Namecheap, etc.) and replace their default nameservers with Cloudflare's nameservers.-

-Manage DNS: You can now add/edit records (A, CNAME, TXT) directly from the Cloudflare DNS dashboard.

6. Difference Between Proxied vs. DNS Only
-Proxied (Orange Cloud 🧡) vs DNS Only (Grey Cloud grey)
-Hides your server's real IP (shows Cloudflare IPs). vs Exposes your server's real IP to the public.
